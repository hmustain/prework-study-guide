CSS

  • Enter your CSS notes here
  • CSS is what gives your website it's unique look and feel
  • Think of CSS as the paint and color of the home building process
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • internal style sheet contains CSS rules for the webpage in the head section of the html file and only apply on that page
  • external style sheet is the most common method which allows developers to keep their rules in a separate file, which makes changes easier

GIT

  • Enter your Git notes here
  • GIT is the free and open source distributed version control system that's responsible for everything GitHub that happens locally on your computer
  • GITHub is where your coding for your website is stored
  • You have one main branch, which is your active live site coding
  • you can have several off branches from the main branch that will allow you to make edits to the code that will not effect the main branch
  • this also allows multiple developers to work on multiple issues at the same time
  • once you have made the necessary updates in an off branch, you can have another developer review your updates
  • once updates have been reviewed, you can merge this off branch back into the main branch
  • this is where your updated coding will take effect into the main live site
  • git codes are always prefaced with git
  • examples include
  • git status:checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
